# University of Michigan
## Masters in Applied Data Science (MADS)

### Milestone I

This project showcases our team's ability to:
 - Acquire Data
 - Clean and Transform Data as part of Data Exploration
 - Analyze Data for trends using standard statistical methods
 - Visualize Data using a variety of techniques
 - Communicate Results in a Written Report

Jupyter notebooks are included in the "src" folder, as well as samples of the datasets we used.
The team primarily used Google Colab for collaboration and development.
